# -Приложението се състои от два контейнера: web (Node.js) и db (MongoDB). Те комуникират помежду си чрез вътрешната мрежа на Docker. Уеб сървърът се свързва към базата данни, използвайки името на услугата db като хост адрес. Връзката е автоматизирана чрез променливата на средата DB_HOST, дефинирана в compose.yml.
